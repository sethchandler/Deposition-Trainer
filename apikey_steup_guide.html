ü§ñ What Are API Keys?

API Key = Your Personal Access Code

Think of an API key like your credit card number - it's a unique code that identifies you and allows you to pay for AI services programmatically.

Key Differences from Regular Subscriptions:

  • ChatGPT Plus subscription ($20/month): Unlimited use through ChatGPT website
  • ChatGPT API access: Pay-per-use through applications like the Deposition Trainer
  • Google Gemini: Only available through API (no consumer subscription)
‚ö†Ô∏è Important: Having a ChatGPT Plus subscription does NOT give you API access. These are completely separate services with different billing.

Why You Need API Keys for the Deposition Trainer

The Deposition Trainer is a specialized application that needs to communicate directly with AI services. Instead of you copy-pasting text back and forth, the API key allows the application to send your questions and receive responses automatically.

üîí Keeping Your API Keys Secure

üö® CRITICAL SECURITY INFORMATION

Your API keys are like credit card numbers. Anyone with access can make charges to your account.

‚úÖ DO:

  • Store keys in a password manager (1Password, LastPass, etc.)
  • Only paste keys into trusted applications like the Deposition Trainer
  • Use different keys for different purposes (you can create multiple)
  • Regularly check your billing dashboard for unexpected charges
  • Delete keys you're no longer using

‚ùå NEVER:

  • Share keys in emails or text messages
  • Post keys in online forums or chat rooms
  • Save keys in unsecured documents
  • Give keys to untrusted applications
  • Use the same key for multiple unrelated projects
üí° If you suspect a key is compromised: Immediately delete it from your provider's dashboard and create a new one.

üîß Troubleshooting Common Issues

Problem: "Invalid API Key" Error

Solutions:

  • Check for typos: API keys are long and easy to mistype
  • Verify the key isn't expired: Check your provider's dashboard
  • Make sure you copied the entire key: They're usually 50+ characters
  • Create a new key: Delete the old one and make a fresh one

Problem: "Quota Exceeded" or "Billing Error"

Solutions:

  • Check your billing: You may have hit your spending limit
  • Add more credit: In your provider's billing section
  • Verify payment method: Credit card may have expired
  • Wait if using free tier: Quotas reset monthly

Problem: "Model Not Available" Error

Solutions:

  • Try a different model: Some models require higher access levels
  • Check your account status: New accounts may have restrictions
  • Wait and retry: Services occasionally have temporary outages

Problem: Slow Responses

Possible causes:

  • High demand: Services can be slower during peak hours
  • Large requests: Very long prompts take more time to process
  • Internet connection: Check your network speed
  • Try a different model: Some are optimized for speed
